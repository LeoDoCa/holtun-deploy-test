# **App Name**: Holtum Essence

## Core Features:

- Landing Page: A visually appealing landing page with a hero section, brand overview, and benefits.
- Product Catalog: A static catalog showcasing product categories and individual product details.
- Category Pages: Display product listings organized by category.
- Product Detail Pages: Detailed pages for each product, showcasing information and imagery.
- Navigation: Simple and intuitive navigation for seamless user experience.
- Footer: A professional footer with essential links and information.

## Style Guidelines:

- Primary color: Soft green (#799E7B) to reflect nature and tranquility.
- Background color: Light, desaturated green (#F2F4F2). Provides a subtle backdrop that complements the primary color without overpowering it.
- Accent color: A muted, earthy brown (#A68B6D). Offers a warm, natural contrast and enhances the sense of wellness.
- Primary font: 'Raleway' (sans-serif) for headings, providing an elegant and professional feel. Note: currently only Google Fonts are supported.
- Secondary font: 'Montserrat' (sans-serif) for body text, ensuring readability and a clean, modern aesthetic. Note: currently only Google Fonts are supported.
- Use minimalist line icons related to essential oils and natural products.
- Implement subtle animations and smooth transitions for a premium user experience.
